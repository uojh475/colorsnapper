## 它到底是干什么用的

一句话：**屏幕上任意位置取色，直接输出你要的代码格式，一键粘贴进编辑器。**

## Photoshop / Illustrator 集成：不需要任何配置

这是我觉得比较加分的一点。

ColorSnapper 2 和 Photoshop、Illustrator 之间有集成，你可以直接从 PS/AI 的当前窗口或者背景窗口取色，并且直接改变前景色/背景色——不需要装插件，不需要改任何设置，装上就能用。

有一个小细节要注意：ColorSnapper 取色走的是**显示器颜色配置文件并转换为 sRGB**，所以从 Adobe 软件取到的颜色可能和 Adobe 自己颜色管理器里显示的有偏差（Adobe 有自己一套颜色管理）。日常开发和网页设计场景，sRGB 是标准，没问题；如果你是印刷设计，用 CMYK 工作流的话，这个需要留意。
